# DisasterSense AI 🚨

DisasterSense AI is a real-time disaster alert chatbot built using Python and Flask. It provides live disaster warnings, safety guidance, and emergency information for India.

## Features
- Real-time disaster alerts
- Earthquake, flood, cyclone, and heatwave guidance
- Emergency helpline numbers
- Location-based alerts
- Interactive chatbot interface

## Technologies Used
- Python
- Flask
- HTML/CSS/JavaScript
- Requests API

## Installation

```bash
pip install -r requirements.txt
python disastersense_ai.py
```

Open in browser:

```text
http://localhost:5000
```
